# PDF 批注截图

本目录保存关键 PDF 批注截图或批注 PDF，用来说明字段来自哪里。

## 用途

- 标注招股书 PDF 中融资事件关键字段的来源位置
- 为 evidence_text 提供可视化的回源追溯
- 方便人工复核时快速定位

## 建议内容

每家公司至少提供以下批注截图:

1. **股本演变概图** — 展示公司设立以来的增资/股权转让/改制路径
2. **融资事件原文** — 标注 total_investment_amount / share_price / investors 的来源段落
3. **估值依据** — 标注 pre_money_valuation / post_money_valuation 的计算来源

## 命名规范

```
{股票代码}_{公司简称}_{批注内容}_{页码}.png

示例:
  301581_黄山谷捷_首次增资_第725页.png
  688758_赛分科技_PreIPO轮增资_第59页.png
```
