# 科创板招股说明书 — 数据来源规则

## 1. 来源平台优先级

| 优先级 | 平台 | 网址 | 覆盖情况 |
|--------|------|------|----------|
| 第一 | 巨潮资讯网 | https://www.cninfo.com.cn | 全部A股IPO，PDF直链稳定 |
| 第二 | 上交所科创板官网 | https://kcb.sse.com.cn | 仅科创板，部分需登录 |
| 第三 | 证监会官网 | http://www.csrc.gov.cn | 预披露文件，可作为补充校验 |

## 2. 巨潮资讯网检索路径（主要来源）

```
首页 → 搜索框输入"公司全称"或"股票代码"
     → 筛选类别：招股说明书
     → 按发布时间倒序
     → 点击进入详情页
     → 复制PDF下载链接（static.cninfo.com.cn 域名）
```

### 搜索关键词规则

- 关键词1：公司全称（如"苏州赛分科技股份有限公司"）
- 关键词2：公司简称（如"赛分科技"）
- 备用关键词：股票代码（如"688758"）

### 链接格式识别

- 有效链接：`https://static.cninfo.com.cn/finalpage/YYYY-MM-DD/数字.PDF`
- 有效期：巨潮PDF直链通常在招股书发布后30天内有效，超期可能失效需重新搜索

## 3. 上交所科创板检索路径（备用来源）

```
上交所科创板 → IPO项目 → 招股说明书
             → 筛选"注册稿"或"正式稿"
             → 按股票代码查找
```

## 4. 文件筛选规则

### 必须包含
- 文件名含"招股说明书"或"招股意向书"
- PDF页数 300-800 页
- 发布者为公司（非交易所/律所）

### 必须排除
- 招股说明书**摘要**
- 招股意向书**摘要**
- 问询函回复 / 问询回复函
- 上市公告书
- 发行公告
- 补充法律意见书
- 审计报告

### 版本优先级
1. 正式稿（上市前最终版，融资历史章节最完整）
2. 注册稿（与正式稿内容几乎一致）
3. 上会稿（可能缺少部分投资人信息）
4. 申报稿（早期版本，可用作交叉校验）

## 5. 科创板特殊说明

- 科创板公司行业集中在半导体、医药、高端制造，PE/VC融资轮次多（平均3-5轮）
- 部分公司存在红筹/VIE架构历史，融资主体可能在境外，需注意招股书中"发行人股本演变"与"重组"章节的区别
- 未盈利企业（标准五）的融资历史通常更长，轮次更多
- 半导体类公司（摩尔线程、沐曦、昂瑞微等）融资金额大，投资方集中度高
