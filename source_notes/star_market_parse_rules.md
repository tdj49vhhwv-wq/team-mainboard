# 科创板 PDF解析规则

## 1. 解析工具

使用项目统一解析模块 `code/04_parse_pdf_to_markdown/parse_pdf.py`。

```bash
cd code/04_parse_pdf_to_markdown
python3 parse_pdf.py --market star --year 2025
```

## 2. 解析参数

| 参数 | 值 | 说明 |
|------|-----|------|
| parser | pymupdf | 主要解析引擎 |
| fallback_parser | pdfplumber | 表格密集页面备用 |
| encoding | utf-8 | 输出编码 |
| table_mode | accurate | 科创板表格多，用高精度模式 |

## 3. 科创板招股书结构特点

科创板招股书与主板/创业板结构一致（统一格式准则），但有以下特点：

### 章节结构（与融资历史相关）
```
第五节 发行人基本情况
  ├── 二、发行人设立情况
  │   ├── （一）设立方式
  │   ├── （二）发起人及出资情况
  │   └── （三）股本演变/历次股本变化  ← 核心章节
  ├── 三、发行人重大资产重组情况
  └── 四、发行人员工持股计划
```

### 科创板特殊段落标记
- "符合科创板定位" / "科创属性" → 描述技术壁垒时出现
- "尚未盈利" / "累计未弥补亏损" → 标准五公司常见
- "特别表决权" / "AB股" → 部分公司有差异化表决安排
- "红筹架构" / "VIE协议" → 涉及境外架构的公司

## 4. 解析质量规则

### Markdown输出要求
- 表格保留：投资人出资额、持股比例等表格必须保留为markdown table
- 层级准确：标题层级不超过4级（# ~ ####）
- 页码标注：每页起始位置标注 `[页XX]`
- 去噪规则：
  - 删除页眉页脚（"科创板招股说明书"重复文字）
  - 删除签字页（无实质内容）
  - 保留所有含数字的表格

### 科创板特殊表格处理
- 股本演变表格：通常含"增资/转让/公积金转增/未分配利润转增"等列
- 股东持股表格：含"股东名称/持股数/持股比例/关联关系"
- 融资轮次表格：部分公司以列表形式列示，需特殊处理

## 5. 解析失败处理

| 问题 | 表现 | 处理 |
|------|------|------|
| 扫描件PDF | 文字层缺失 | 标记为OCR需求，暂跳过 |
| 表格断裂 | 跨页表格行错位 | 手动拼接后重新解析 |
| 编码异常 | 公司名/人名乱码 | 用pdfplumber备用引擎重试 |
| 超大文件 | 800+页 | 截取关键章节（第五节）单独解析 |

## 6. 解析后校验

- [ ] Markdown文件总行数 > 500
- [ ] 第五节"发行人基本情况"完整保留
- [ ] 表格数量与原PDF一致（±2）
- [ ] 无乱码段落
- [ ] 记录到 parse_log.csv
