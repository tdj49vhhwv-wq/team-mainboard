# 科创板 章节定位规则

## 1. 定位目标

从招股书Markdown中找到包含融资历史（增资/股权转让/PEVC投资）的章节段落。

## 2. 目标章节

### 主目标：第五节 发行人基本情况 → 股本演变
```
第五节 发行人基本情况
  └── 二、发行人设立情况
       ├── （一）设立方式         ← 含初始出资信息
       ├── （二）发起人            ← 含发起人出资比例
       └── （三）股本/股权演变     ← ★ 核心：历次增资+股权转让
```

### 辅目标：其他可能含融资信息的章节
```
第四节 风险因素
  └── 与发行人相关的风险           ← 偶尔提及PEVC对赌条款

第六节 业务与技术
  └── 核心技术人员的持股情况       ← 含员工持股平台信息

第十节 投资者保护
  └── 特别表决权/差异表决权安排    ← 含特殊股东权利
```

## 3. 章节定位关键词

### 一二级标题匹配（按优先级）
1. **"股本"** + **"演变"/"变化"/"形成"/"变动"** → 首选
2. **"股权"** + **"演变"/"变化"/"变动"/"结构"** → 次选
3. **"增资"** + **"扩股"/"情况"** → 辅助
4. **"发行人"** + **"设立"/"重组"/"改制"** → 补充
5. **"股东"** + **"出资"/"持股"** → 补充

### 科创板特殊关键词
- "历次增资及股权转让" → 最完整的融资历史描述
- "股本形成及变化" → 较完整的版本
- "发行人股本演变" → 标准表述
- "红筹架构拆除" / "VIE解除" → 部分公司涉及境外重组

## 4. 定位算法

```python
# 伪代码
def locate_sections(markdown_content):
    # Step 1: 定位"第五节"或"发行人基本情况"
    section_5 = find_heading(md, r"第[五5]节.*发行[人人]基本情况")
    
    # Step 2: 在第五节范围内定位"股本演变"子章节
    within_s5 = extract_range(md, section_5.start, next_major_section)
    equity_section = find_heading(within_s5, r"(股本|股权).*(演变|变化|形成|变动)")
    
    # Step 3: 截取到下一个同级标题之前
    result = extract_to_next_sibling(equity_section)
    
    # Step 4: 降级搜索（如果没找到）
    if not result:
        result = fallback_search(md, keywords=["增资", "股权转让", "出资方式"])
    
    return result
```

## 5. 科创板定位注意事项

### 5.1 标准五公司（未盈利医药企业）
- 必贝特、禾元生物等可能无"增资"记录，融资以"股权转让"和"可转债/可转换优先股"形式存在
- 需扩展到"与股东之间的特殊安排"章节

### 5.2 半导体/硬科技公司
- 摩尔线程、沐曦、昂瑞微等融资轮次多（A/B/C/D轮），多轮次表述可能分散在多个段落
- 定位范围可能需要扩展到"子公司股本演变"（部分融资发生在子公司层面）

### 5.3 有红筹历史的公司
- 影石创新等可能有VIE架构拆除历史
- 融资历史可能分布在"红筹架构搭建"和"红筹架构拆除"两个段落
