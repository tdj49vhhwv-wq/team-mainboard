# 科创板 PDF下载规则

## 1. 下载脚本

使用项目统一下载模块 `code/03_download_pdfs/downloader.py`，无需市场特殊定制。

```bash
cd code/03_download_pdfs
python3 downloader.py --market star --year 2025
```

## 2. 下载参数规范

| 参数 | 值 | 说明 |
|------|-----|------|
| market | star | 科创板标识 |
| year | 2025 | 仅处理2025年上市公司 |
| source | cninfo | 默认使用巨潮资讯 |
| fallback | sse | 巨潮失效时回退上交所官网 |

## 3. 文件命名规则

```
{股票代码}_{公司简称}_招股书_{版本}_{日期}.pdf

示例：
688758_赛分科技_招股书_正式稿_20250106.pdf
688795_摩尔线程_招股书_正式稿_20251128.pdf
```

- 股票代码：6位数字
- 公司简称：不含"股份有限公司"
- 版本：正式稿 / 注册稿 / 上会稿
- 日期：YYYYMMDD格式，取自招股书签署日期

## 4. 存储路径

```
data/star_market/pdf/
├── 688758_赛分科技_招股书_正式稿_20250106.pdf
├── 688583_思看科技_招股书_正式稿_20250110.pdf
├── ...
└── 688809_强一股份_招股书_正式稿_20251225.pdf
```

## 5. 下载失败处理

| 失败类型 | 原因 | 处理方式 |
|----------|------|----------|
| 链接过期 | 巨潮直链30天有效期 | 重新在巨潮搜索获取新链接 |
| 站点反爬 | 频繁请求被限 | 间隔30s-60s重试，最多3次 |
| 文件不存在 | 公司更名/代码变更 | 用公司全称重新搜索 |
| 网络超时 | 服务器无响应 | 切换上交所科创板备用源 |

## 6. 下载后校验

- [ ] PDF文件大小 > 1MB（排除空文件/错误页）
- [ ] PDF页数在 300-800 范围内
- [ ] PDF可正常打开，不含密码保护
- [ ] 文件命名符合规则
- [ ] 记录到 download_log.csv
